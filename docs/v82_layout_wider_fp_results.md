# v82 wider layout and Field Potential results placement

- Main horizontal splitter now starts near 50/50, giving the left controls panel roughly the same width as the plot/results side.
- The controls panel minimum width is increased to reduce cramped Field Potential controls.
- The bottom results area now has tabs:
  - Detection
  - Field Potential
- Field Potential results moved out of the left subpanel and into the larger bottom results area.
- Field Potential tab now contains compact settings/actions only, with a note pointing to the bottom Field Potential results tab.
- Running FP analysis switches the bottom results area to Field Potential.
- Running event detection switches the bottom results area to Detection.
