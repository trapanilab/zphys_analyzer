# v62 Windows panel millisecond controls

- Millisecond controls in the Windows subpanel now display whole numbers:
  onset, first width, second gap, and second width.
- The period/half-period readout now displays whole milliseconds.
- The second-window width starts matched to the first-window width but remains
  editable.
- The first-window width no longer continuously locks the second-window width;
  frequency/default changes still set the second width as a starting point.
