# v72 Analysis Tools reorganization

The Analysis Tools tab now shows the Detect / Windows / Actions subtabs directly
at the top. The former top-level Analysis setup box is now nested inside the
Detect subtab as "Detection setup".

Detect tab order:
1. Detection setup
2. Threshold detection
3. Window shortcuts (shared with the Windows tab)

Small label clarifications:
- "Analyze" -> "Analyze source"
- "Selected" -> "Selected sweeps"
- "Inside windows" -> "Restrict to windows"
