# v29 manual frequency entry

Removed the `Frequency preset` dropdown from the Windows tab.

The frequency workflow is now manual and direct:

- enter the frequency in `Frequency (Hz)`
- the period readout updates immediately
- `Generate Stimulus Windows` uses the manual frequency value

The `Set 20 Hz / 50 ms Defaults` button remains as a convenience shortcut, but
there is no longer a separate preset dropdown state that can get out of sync.
