# v35 File Tools, window shortcut labels, threshold line, and tab scrolling

## File Tools

`Show Loaded Metadata` and `Where Is Data Stored?` now sit side-by-side with
explanatory text below them.

## Window shortcuts

The shortcuts are clarified:

- `Add Manual Window`: adds one window from the current view/cursor-style region.
- `Generate Stimulus Windows`: creates repeated windows from onset/frequency/count.

## Threshold line

The plotted threshold line is thicker for visibility.

## Scrolling

The outer left control panel no longer scrolls as one unit. Each tab contains
its own scroll area so the top header/common controls stay fixed while only the
relevant tab contents scroll.
