# v44 signal autoscale, narrow shortcut buttons, and stimulus right axis

## Signal switching

Switching between S1/S2/All now autoscale-ranges the new plot immediately.

## Window shortcuts

The narrow Detect-tab shortcut buttons are shortened:

- Add
- Gen
- Clear

Tooltips retain the full meaning.

## Stimulus overlay

Stimulus overlay now uses a separate right Y axis linked to the same X/time axis.
This avoids the stimulus trace disappearing when stimulus and recording signals
have very different amplitudes/units.

S2 remains a recorded input channel and is still plotted separately in blue.
