# v28 compact top panel

The top of the left control panel was tightened and made more visually prominent.

Changes:

- `Load File` is now styled as a blue primary button.
- Common actions use a compact grid instead of stacked rows.
- Analysis setup uses a compact grid.
- Long explanatory text was moved into tooltips.
- Window summary is constrained in height to save vertical space.

All prior features remain available.
