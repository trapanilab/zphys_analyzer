# Release 0.1.84 portable package

Packaging additions:
- `START_HERE.txt`
- `INSTALL.md`
- `requirements.txt`
- `environment.yml`
- `scripts/install_mac.command`
- `scripts/run_mac.command`
- `scripts/install_windows.bat`
- `scripts/run_windows.bat`
- `scripts/install_conda_mac.command`
- `scripts/install_conda_windows.bat`
- `launch_zphys.py`
- `launch_zphys.pyw`

The package can be installed with either:
- local Python virtual environment scripts, or
- conda/Miniforge environment scripts.

The package remains source-based rather than a signed binary `.app` or `.exe`.
