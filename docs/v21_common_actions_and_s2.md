# v21 common actions and S2 matching

- Autoscale and Baseline Subtract are now in a Common actions panel above the tabs.
- The baseline all-selected-sweeps option is also in Common actions.
- `Load Data`/`Load File` wording is standardized to `Load File`; both meant opening a file chooser and loading a recording.
- S2/stimulus matching now tries exact routine matching, routine-name matching, S1-to-S2 name substitution, then shows a chooser of all S2-like series if needed.
