# v18 UI polish

## Window defaults

The default number of generated windows is now 1.

## Baseline controls

The checkbox controlling whether baseline subtraction applies to all selected
sweeps in all-series scope now lives next to the Baseline Subtract button in the
Actions tab.

## Window shortcut buttons

The Detect tab window shortcut labels are shortened so they fit better:

- Add Window
- Generate
- Clear

Tooltips explain the full meaning.
