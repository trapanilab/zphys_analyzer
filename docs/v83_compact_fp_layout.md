# v83 compact Field Potential layout

- Main horizontal splitter now starts at approximately a 1/3 left, 2/3 right ratio.
- Left panel minimum width reduced from 520 px to 360 px so the drag-stop/startup width matches the preferred compact ratio.
- Field Potential numeric settings are more compact:
  - time fields show 3 decimals instead of 6
  - RMS tolerance shows 2 decimals
  - spin boxes have fixed narrow widths
- Field Potential settings are arranged two per row where possible.
