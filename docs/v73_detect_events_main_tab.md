# v73 Detect Events main tab

Detect Events is now a fourth main tab.

Main tabs:
- File Tools
- Detect Events
- Analysis Tools
- Display Tools

Detect Events contains:
- A top Quick detect box with Find Spikes / Events always visible.
- Detect and Windows subtabs beneath it.
- Detection setup, threshold controls, advanced detection settings, and window shortcuts.

Analysis Tools now focuses on trace-processing actions such as baseline,
averaging, concatenation, and FFT.
